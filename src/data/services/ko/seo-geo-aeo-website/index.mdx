## 왜 일반 홈페이지로는 부족한가

수백만 원 들여 만든 홈페이지가 구글 검색에 나오지 않습니다. ChatGPT에 물어봐도 당신의 비즈니스를 모릅니다. 디자인은 예쁘지만, 검색 엔진과 AI가 이해할 수 없는 구조로 만들어졌기 때문입니다.

## Thales Web이 다른 점

Astro 프레임워크를 기반으로 SEO, GEO, AEO 3가지 최적화가 처음부터 내장된 홈페이지를 구축합니다.

### SEO (검색 엔진 최적화)
- Lighthouse 성능 점수 98+
- Core Web Vitals(LCP, INP, CLS) 완벽 통과
- 시맨틱 HTML, 자동 sitemap/RSS 생성
- 정적 사이트 생성(SSG)으로 초고속 로딩

### GEO (생성형 엔진 최적화)
- GPTBot, ClaudeBot, PerplexityBot 크롤러 접근 허용
- JSON-LD 구조화 데이터 자동 생성 (Organization, BlogPosting, Person, FAQ)
- CitationCapsule 컴포넌트로 AI 인용 가능 콘텐츠 마크업
- llms.txt 자동 생성으로 AI 모델에 사이트 구조 안내

### AEO (답변 엔진 최적화)
- FAQ 스키마 자동 생성
- TldrBlock, KeyTakeaways로 직접 답변형 콘텐츠 구조
- SpeakableSpecification 마크업 적용

### 직접 관리 가능
- Keystatic CMS 내장 — 코딩 없이 블로그, 프로젝트, FAQ 직접 수정
- 관리자 페이지(/admin)에서 클릭만으로 콘텐츠 관리
