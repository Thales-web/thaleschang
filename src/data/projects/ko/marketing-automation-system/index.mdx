## 프로젝트 개요

중소기업과 1인 기업을 위한 마케팅 자동화 시스템입니다. 웹사이트 방문자가 문의 폼을 작성하는 순간부터 고객이 되기까지의 전체 여정을 자동화합니다. 수작업을 최소화하고, 비즈니스 성장에 집중할 수 있게 합니다.

## 자동화 워크플로우

### 리드 파이프라인
1. 웹사이트 문의 폼 접수
2. CRM(Supabase)에 자동 등록
3. 환영 이메일 즉시 발송
4. 3일/7일/14일 후속 이메일 시퀀스 자동 진행

### 콘텐츠 자동 배포
1. 블로그 글 발행 감지
2. 각 SNS 채널(LinkedIn, Twitter, Facebook)에 최적화된 포맷으로 자동 공유
3. IndexNow API로 검색 엔진에 즉시 알림

### 성과 추적
- Google Analytics, Naver Analytics 데이터 통합
- 리드 전환율, 이메일 오픈율, SNS 참여율 실시간 모니터링

## 기대 효과

- 리드 응답 시간 48시간 → 15분 이내
- 수작업 마케팅 업무 70% 감소
- 리드-고객 전환율 향상
