## 프로젝트 개요

검색의 패러다임이 변하고 있습니다. 구글 검색(SEO)만으로는 충분하지 않습니다. ChatGPT, Perplexity 같은 AI가 답변할 때 인용되는 것(GEO), 음성 검색과 Featured Snippet에 선택되는 것(AEO)까지 — 세 가지 검색 채널 모두에 최적화된 웹사이트를 구축합니다.

## SEO 최적화

- Lighthouse 성능 점수 98+, 접근성 100
- 시맨틱 HTML 구조, 자동 sitemap/RSS 생성
- Core Web Vitals 최적화 (LCP, INP, CLS)
- 정적 사이트 생성(SSG)으로 초고속 로딩

## AEO 최적화

- FAQ 스키마(JSON-LD) 자동 생성
- 질문-답변 형식의 콘텐츠 구조
- SpeakableSpecification으로 음성 검색 대응

## GEO 최적화

- GPTBot, ClaudeBot, PerplexityBot 크롤러 접근 허용
- CitationCapsule 컴포넌트로 AI 인용 가능 콘텐츠 마크업
- llms.txt 자동 생성으로 AI 모델에 사이트 구조 안내
- Person/Organization 스키마의 sameAs로 엔티티 인식 강화
