## 프로젝트 개요

콘텐츠 마케팅의 가장 큰 병목은 '꾸준한 콘텐츠 생산'입니다. 이 시스템은 키워드 입력만으로 SEO/GEO 최적화된 블로그 글을 자동 생성하고, 검수 후 원클릭으로 발행할 수 있는 파이프라인입니다.

## 자동화 워크플로우

1. **키워드 입력**: 타겟 키워드와 주제를 입력
2. **AI 콘텐츠 생성**: OpenAI API가 SEO 최적화된 콘텐츠 초안 작성
3. **메타데이터 자동 설정**: 제목, 메타 설명, 태그, 카테고리 자동 생성
4. **이미지 생성**: 히어로 이미지 자동 생성 및 최적화
5. **검수 및 발행**: 관리자 검수 후 GitHub API를 통해 Astro 블로그에 자동 커밋
6. **자동 배포**: Netlify가 변경 감지 후 자동 빌드 및 배포
7. **SNS 공유**: 발행된 글을 각 SNS 채널에 최적화된 포맷으로 자동 공유

## 기대 효과

- 콘텐츠 생산 시간 80% 단축
- 일관된 SEO/GEO 최적화 품질 유지
- 주 1회 → 주 3~5회 발행 빈도 향상
